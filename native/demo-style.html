Video.js - 4种主题(sea、city、forest、fantasy)

video.js：https://videojs.com， gitHub：https://github.com/videojs/themes

默认主题：

sea主题：

city主题：

forest主题：

fantasy主题：

        
        一、直接在html文件中的head中引入

            
            <link href="https://unpkg.com/video.js@7/dist/video-js.min.css" rel="stylesheet">
            
            
            <link href="https://unpkg.com/@videojs/themes@1/dist/city/index.css" rel="stylesheet">
            
            
            <link href="https://unpkg.com/@videojs/themes@1/dist/fantasy/index.css" rel="stylesheet">
            
            
            <link href="https://unpkg.com/@videojs/themes@1/dist/forest/index.css" rel="stylesheet">
            
            
            <link href="https://unpkg.com/@videojs/themes@1/dist/sea/index.css" rel="stylesheet">
       


        二、在JavaScript中使用CSS模块，则可以安装NPM模块：

            npm install --save video.js @videojs/themes
        
            //然后，只需像导入其他CSS一样导入文件即可。
       
            import 'video.js/dist/video-js.css';
            
            // City
            import '@videojs/themes/dist/city/index.css';
            
            // Fantasy
            import '@videojs/themes/dist/fantasy/index.css';
            
            // Forest
            import '@videojs/themes/dist/forest/index.css';
            
            // Sea
            import '@videojs/themes/dist/sea/index.css';
            

            在video标签有class属性中 vjs-theme-$ {THEME_NAME}，因此vjs-theme-city代表城市主题，而vjs-theme-sea代表sea主题。

            <video id="my-player" class="video-js vjs-theme-city" > </video>
